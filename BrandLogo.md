# Task: Design and Implement a Professional Brand Identity for StayOra

You are an expert Brand Designer, UI Designer, and Frontend Engineer.

Your mission is to create a premium, modern, memorable, and scalable brand identity for the travel booking platform "StayOra", then integrate it throughout the project.

## About StayOra

StayOra is an all-in-one travel and booking platform.

Users can book:

- Hotels
- Apartments
- Resorts
- Shared Rooms
- Convention Halls
- Transport
- Tours
- Activities
- Visa Services

The brand should feel:

- Premium
- Trustworthy
- Modern
- Clean
- Friendly
- Global
- Elegant
- Minimal

Avoid looking like a generic hotel logo.

The logo should work for a travel ecosystem, not just hotels.

# Logo Goals

Create a logo that:

- instantly feels premium

- is scalable

- looks great in both light and dark mode

- remains timeless
